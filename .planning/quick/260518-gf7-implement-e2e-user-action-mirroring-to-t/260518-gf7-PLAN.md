## Goal
Add `logUserAction` to `e2eSender` and wire it into `botRig.inject` so every
injected user message also appears in the real Telegram chat as `👤 <text>`.

## Tasks

### T1 — Add `logUserAction` to `e2eSender`

**File:** `internal/e2e/helpers_test.go`

Add after `AckCallback`:

```go
func (s *e2eSender) logUserAction(text string) {
    _ = s.real.Send("👤 " + text)
}
```

**verify:** method exists, sends to `s.real` only (no `s.out <- ...`)
**done:** method visible in file

### T2 — Call `logUserAction` inside `botRig.inject`

**File:** `internal/e2e/helpers_test.go`

Add as first line of `inject` body:

```go

```

**verify:** `inject` calls `r.sender.logUserAction(text)` before building the update
**done:** call present in inject body
