<objective>
Implement `internal/handler.QuestionFlow` â€” the state machine that sends a question, accepts
the user's text reply as the answer, advances the session, sends the next question, and
finalises (write completed entry + delete session) after the last one. Persistence after
every answer (SESS-01) and immediate next-question on advance (BOT-06).
</objective>

    <truth>SendQuestion(slug, idx) renders questions[idx]. If `Example` is non-empty, emit the question on one line, then `_Example: <text>_` on the next (Markdown italic).</truth>
    <truth>HandleAnswer(slug, text, now) records the answer via session.Manager.RecordAnswer, then either sends the next question OR finalises (writes a completed entry, deletes session.yaml, sends `âś… <Name> complete! Answers saved.`).</truth>
    <truth>FinalizeIfDone(slug, now) checks if the in-memory session at `len(questions)` is fully answered; if so, finalises. Used by the startup restore path (03-03).</truth>
    <truth>All time values come from a `Now func() time.Time` field on QuestionFlow so tests can pin time. Default is `time.Now`.</truth>

<task id="03-02-T02">
  <title>SendQuestion + HandleAnswer + Finalize</title>
  <read_first>
    - internal/handler/flow.go (just created)
    - .planning/REQUIREMENTS.md (BOT-03..07)
    - original-prd.md (US-006, US-007)
  </read_first>
  <action>
    Extend `internal/handler/flow.go`:

    - `func (f *QuestionFlow) StartQuestionnaire(slug string, scheduled time.Time) error`:
      Looks up the Questionnaire by slug. Calls `f.Sessions.Start(slug, scheduled, f.Now(), q.Location)`. Then calls `f.SendQuestion(slug, 0)`.

    - `func (f *QuestionFlow) SendQuestion(slug string, idx int) error`:
      Looks up Questionnaire by slug. If idx >= len(questions), returns nil (caller should have finalised). If `Example == ""`, calls `Sender.Send(question)`. Otherwise `Sender.SendMarkdown(question + "\n_Example: " + example + "_")`.

    - `func (f *QuestionFlow) HandleAnswer(slug, text string) error`:
      1. Get current session via `f.Sessions.Get(slug)`. If nil, return an error.
      2. q := Questionnaires[slug]; current := questions[session.CurrentQuestionIndex].
      3. Call `f.Sessions.RecordAnswer(slug, current.Question, text)`.
      4. Re-read session. If `CurrentQuestionIndex < len(questions)`, call `SendQuestion(slug, CurrentQuestionIndex)` and return.
      5. Otherwise call `f.finalize(slug, session)`.

    - `func (f *QuestionFlow) finalize(slug string, s *session.Session) error`:
      1. Parse `s.ScheduledFor` back into time.Time via `time.Parse(time.RFC3339, ...)`.
      2. Build []storage.AnswerPair from s.Answers.
      3. `storage.PrependCompleted(f.DataDir, slug, scheduled, f.Now(), q.Location, answers)`.
      4. `f.Sessions.Delete(slug)`.
      5. `f.Sender.Send("âś… " + q.Name + " complete! Answers saved.")`.

    - `func (f *QuestionFlow) FinalizeIfDone(slug string) (bool, error)`:
      Reads session. If `CurrentQuestionIndex >= len(questions)`, calls finalize and returns true.
  </action>
  <acceptance_criteria>

    - For a question with Example, the sender's SendMarkdown was called with text containing the literal `_Example: ` substring

    - After HandleAnswer is called for the LAST question, the sender receives a completion message containing the questionnaire's Name and the literal `âś…`, and session.yaml is deleted
    - After finalise, `answers.yaml` contains a completed entry with the recorded answers in order
    - FinalizeIfDone on a session at CurrentQuestionIndex==len(questions) returns true and removes session.yaml
    - FinalizeIfDone on an incomplete session returns false with no error
