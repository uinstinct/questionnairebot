<objective>
Implement `internal/session.Manager` — an in-memory registry mapping `slug -> *Session`
protected by a single `sync.Mutex`. Save writes `data/<slug>/session.yaml` atomically after
every recorded answer; Delete removes it idempotently; LoadFromDisk rehydrates from an
existing `session.yaml` on startup. The smoke test must pass `go test -race`.
</objective>

<must_haves>
  <truths>
    <truth>One `sync.Mutex` per Manager guards both the in-memory map AND each session's disk write. All exported methods take/release the lock.</truth>
    <truth>session.yaml schema (PRD §4.3): `questionnaire_id`, `scheduled_for`, `started_at` (both RFC3339 with offset), `current_question_index` (int, 0-based, points to NEXT unanswered question), `answers` ([]AnswerPair).</truth>
    <truth>Save uses temp+rename like storage.Prepend: write `<path>.tmp`, fsync, rename to `<path>`. Crash mid-write leaves the prior `session.yaml` intact.</truth>
    <truth>Delete: `os.Remove`; ignore `os.ErrNotExist` so the call is idempotent.</truth>
    <truth>RecordAnswer appends to the session's answers slice, increments `current_question_index`, and calls Save — under the lock.</truth>
    <truth>LoadFromDisk(slug, dataDir) returns (nil, nil) when no session.yaml exists. When it exists, parse and register the session in the manager.</truth>
  </truths>
</must_haves>

<task id="02-02-T02">
  <title>Manager with Save/Delete/LoadFromDisk/RecordAnswer/Get/Start</title>
  <read_first>
    - internal/session/types.go (just created)
    - internal/storage/storage.go (atomic-write pattern to mirror)
    - .planning/REQUIREMENTS.md (SESS-01..03)
  </read_first>
  <action>
    Create `internal/session/manager.go` with:
    - `type Manager struct { mu sync.Mutex; sessions map[string]*Session; dataDir string }`
    - `func NewManager(dataDir string) *Manager`.
    - `func (m *Manager) Start(slug string, scheduled, started time.Time, loc *time.Location) (*Session, error)`:
      Builds Session{QuestionnaireID: slug, ScheduledFor: scheduled.In(loc).Format(RFC3339), StartedAt: started.In(loc).Format(RFC3339), CurrentQuestionIndex: 0}, stores in map, writes to disk.
    - `func (m *Manager) Get(slug string) *Session`: returns a *copy* of the session (or nil) so callers can't mutate registry state without going through RecordAnswer.
    - `func (m *Manager) RecordAnswer(slug, question, answer string) error`: appends `{question, answer}` to `m.sessions[slug].Answers`, increments `CurrentQuestionIndex`, persists via internal `saveLocked(slug)`. Returns an error if no session for slug.
    - `func (m *Manager) Delete(slug string) error`: removes from map and from disk (`os.Remove`, ignore ErrNotExist).
    - `func (m *Manager) LoadFromDisk(slug string) (*Session, error)`:
      Reads `<dataDir>/<slug>/session.yaml`; if missing returns (nil, nil); on parse success stores in map.
    - Internal `saveLocked(slug string) error`: marshal session to yaml, atomic write to `<dataDir>/<slug>/session.yaml` via `<path>.tmp` + fsync + rename.
    All exported methods acquire `m.mu` before touching `m.sessions` or disk.
  </action>
  <acceptance_criteria>
    - `internal/session/manager.go` exports Manager with Start, Get, RecordAnswer, Delete, LoadFromDisk
    - `NewManager("./data")` returns a non-nil Manager with empty sessions map
    - After Start("x", t1, t2, UTC), a file `./data/x/session.yaml` exists containing `questionnaire_id: x` and `current_question_index: 0`
    - After RecordAnswer("x", "Q1", "A1"), the disk session.yaml has `current_question_index: 1` and an `answers:` block with one item
    - Delete("x") removes the file and removes "x" from the map
    - Calling Delete("x") twice does not return an error
    - LoadFromDisk on a slug with no session.yaml returns (nil, nil)
    - LoadFromDisk after a Start+RecordAnswer cycle decodes the previously-saved state correctly
    - `go vet ./internal/session/...` exits 0
  </acceptance_criteria>
</task>
